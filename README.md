# CIDR Block 

A simple library for handling and extracting data from CIDR Blocks in order to organize subnets.

## Installation

Documentation can be generated with [ExDoc](https://github.com/elixir-lang/ex_doc)
and published on [HexDocs](https://hexdocs.pm). Once published, the docs can
be found at [https://hexdocs.pm/cidr](https://hexdocs.pm/cidr).
